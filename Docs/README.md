## Get started

Requirements:

* For Windows: `.NET 4.7.2`
* For Linux or macOS: `.NET Core 3.0`

* [Get Lanchat](https://github.com/tofudd/lanchat/releases)
* [Get .NET Runtime for Linux and macOS](https://dotnet.microsoft.com/download/dotnet-core/3.1)
