Lan-chat and file transfer
API

Lanchat uses JSONs with one root key which specifies the type of value.

For example:

{
"Message":
{
"Content": "HaQ\u002BpFLc\u002Bv6R2cjKP\u002BpB1A==",
"Private": false
}
}
{
"NodesList":
[
"192.168.1.3",
"192.168.1.4"
]
}
